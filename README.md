# Итоговая работа по дисциплине ООП - "Менеджер личных финансов" 

Это консольное приложение на Java, которое помогает следить за своими финансовыми средствами. 
Программа позволяет записывать доходы и расходы, устанавливать бюджеты на разные категории и переводить деньги другим пользователям.

Все данные автоматически сохраняются в файл users_data.json, поэтому ничего не потеряется после закрытия программы.

### Что умеет программа

Вход и регистрация: У каждого пользователя свой логин и пароль.
Учет финансов: Можно добавлять доходы и траты, указывая категорию (например, "Еда", "Транспорт").
Бюджет: Можно установить лимит трат на категорию. Если вы потратите больше, программа предупредит об этом.
Статистика: Показывает общий баланс, сколько потрачено по категориям и сколько осталось бюджета.
Переводы: Можно отправить виртуальные деньги другому пользователю по его логину.

### Какие технологии использовались

Java 17 — язык, на котором написана программа.
Maven — для сборки проекта.
Jackson — библиотека, чтобы сохранять данные пользователей в JSON-файл.
JUnit 5 — для тестов. 
GitHub Actions — настроена автоматическая сборка и проверка кода.

### Автоматическая сборка

В репозитории настроен GitHub Actions. 
Каждый раз, при обновлении или загрузке нового кода на GitHub (push), сервер автоматически собирает проект и запускает все тесты. Это гарантирует, что новые изменения ничего не сломали.

Вы можете увидеть статус сборки (зеленую галочку или красный крестик) во вкладке Actions в репозитории.

### Список команд

Управление программой происходит через ввод цифр в консоли.

Команды при запуске (Меню авторизации):
1 — Войти (если у вас уже есть логин и пароль).
2 — Регистрация (создать нового пользователя).
3 — Выход (полностью закрыть программу).

Команды в главном меню:
1 — Добавить доход: программа попросит ввести сумму и категорию. 
2 — Добавить расход: вводите сумму и категорию, а программа проверяет бюджет
3 — Установить бюджет: задать лимит трат для определенной категории.
4 — Статистика: показать текущий баланс, общие доходы и расходы по категориям.
5 — Перевод: отправить деньги другому пользователю (нужно знать его логин).
6 — Выйти: выйти из аккаунта и вернуться в меню авторизации.

### Как запустить проект

### Через консоль 
mvn clean package
java -cp target/MyFinances-1.0-SNAPSHOT.jar Main

### Через IDEA
Запустить класс Main.java

### Тесты
Запуск unit-тестов - mvn test

### Структура проекта

src/main/java/model — здесь лежат основные классы (Пользователь, Кошелек).
src/main/java/service — здесь вся логика (подсчеты, сохранение в файл, авторизация).
src/main/java/cli — классы для работы с консолью (меню и ввод данных).
Main.java — главный файл для запуска.
